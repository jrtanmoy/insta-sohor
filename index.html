1) How does javascript work?

Answer:
Javascript is a single threaded, high level & interpreted programming language. When we writte some code and want to run then it works in an intersting way with the combination of browser. Every browser has different engine like chrome browser use v8 engine to execute javascript. However, when we run the code then firstly the code split into key&value pair and store in the memory. Then its reading & execute the code synchromonously. Synchromonous means every statement of the code gets executed one by one. So, basically a statement has to wait for the earlier statement to get executed. After finishing all the exetution, with the help of browser it's show the execution result in the UI.

3) Difference between local storage and session storage?

Answer:
In local storage the date will remain until we cleared the data but data in session storage is cleared when the page session ends. We can get local storage and session storage data by using getItem(). We can also set both of the storage data by using setItem(). When we want to set object in the storage we have to stringify the object. And also when we want to read the object in the storage we have to parse the object.
